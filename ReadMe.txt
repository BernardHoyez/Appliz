Chers amis marcheurs,

Après le site web V.A.R. que vous connaissez déjà et que vous pouvez consulter depuis votre ordinateur ou votre smartphone, voici une autre option de liaison entre nous. 
Il s'agit d'une application destinée à tous les possesseurs d'un smartphone Android. Désolé pour les possesseurs d'un i-phone Apple, mais la politique commerciale d'Apple oblige que toute application i-OS ne peut être distribuée que par le truchement de l'Apple Store. Ce n'est pas le cas d'Android. La liberté est laissée aux développeurs de ne pas passer par le Play Store Google.

L'application VAR Rando n'a que quelques fonctionnalités assez basiques, mais elle pourra s'enrichir au cours du temps si vous le souhaitez.

1) Les programmes des randonnées du mercredi, du vendredi et du dimanche.
Ce n'est qu'une simple redirection vers les pages concernées du site web.

2) Ma localisation sur une carte Open Street Map. L'épingle rouge indique votre position. Les coordonnées géographiques, latitude et longitude sont affichées.

3)  Une redirection vers la page d'accueil du site VAR. Toutes les pages peuvent être consultées.

4) Une carte zoomable avec des épingles indiquant quelques-unes des randonnées passées.

5) Une carte avec un fond Google Maps permettant de placer un repère soit par ses coordonnées géographiques, soit par une adresse, soit par un code postal. Elle permet également de voir sa propre localisation.

6) Une page sur la sécurité. Comment appeler le 112, réaliser les premiers secours ou s'informer sur les incendies.

7) Une page sur diverses informations sur les circuits de randonnée, les logiciels de randonnée et les circuits touristiques.

8) Une page permettant d'envoyer un SMS à un contact téléphonique l'informant de votre position géographique. Cette option permet de vous retrouver si vous êtes perdu ou si vous avez besoin de secours en cas d'accident.

9) Une page permettant d'enregistrer la trace de son parcours au format KML. Le fichier de cette trace est sauvegardé dans le dossier :
 Android > Data > AppInventor.ai_bhoyez.VAR_version_nouvelle.
Prenez un explorateur de fichiers pour y parvenir.
Vous pourrez visualiser cette trace dans Google Earth ou dans un logiciel de randonnée.
 Comment installer cette application :

1) Cette application est sous la forme d'un fichier d'installation. Les fichiers d'installation sont suffixés .apk.
Le fichier est appelé Var_rando_version_nouvelle.apk
2) Le fichier est déposé sur le site Github. Ce site est utilisé par les développeurs pour partager leurs applications.
3) Télécharger le fichier choisi (regarder la version la plus récente).
Il suffit de cliquer sur la petite flèche dirigée vers le bas.
4) Placer le fichier .apk dans n'importe quel dossier de votre téléphone. Par exemple dans le dossier téléchargement (Download). Si vous avez téléchargé le fichier avec votre ordinateur, branchez votre téléphone avec un câble USB. Dans la notification, choisissez l'option Transfert de fichiers.
5) Avec l'explorateur de fichiers de votre téléphone, mettez-vous dans le dossier contenant l'apk. Cochez sur le nom du fichier. Choisissez l'option : 3 petits points > Ouvrir avec. Choisissez Programme d'installation kit, une seule fois.

6) Le programme va s'installer. Vous aurez certainement plusieurs avertissements de sécurité. C'est normal. Mais dans ce cas, il n'y a pas de danger.

7) Finalement, l'icône apparaît dans la liste des applications déjà installées.
